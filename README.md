# 📖 About The Project

Managing transportation information manually can become difficult as the number of students increases.

This project provides a **simple command-line Transportation Management System** that allows administrators to register students, manage transport information, search records, delete records, and permanently store data using text files.

The project demonstrates fundamental concepts of **C Programming** and serves as a beginner-friendly database management application.
